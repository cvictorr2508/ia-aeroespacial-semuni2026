# Datasets

Este diretório não armazena bases de terceiros por padrão.

A política do projeto é:

1. manter links para repositórios oficiais;
2. fornecer instruções de acesso e scripts de download quando permitido;
3. registrar a licença original de cada dataset;
4. evitar redistribuição indevida.

Datasets previstos para uso didático:

- NASA Airfoil Self-Noise;
- UIUC Airfoil Data Site;
- NPARC Alliance / NASA CFD resources;
- ShapeNet / ShapeNet Part;
- MIPLIB;
- MILPBench e benchmarks de facility location.
