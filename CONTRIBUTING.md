# Contribuindo

Contribuições são bem-vindas, especialmente correções, sugestões de referências, melhorias de notebooks e propostas de novos exemplos didáticos.

## Tipos de contribuição

- Correção de erros conceituais ou tipográficos;
- melhoria de explicações;
- inclusão de referências acadêmicas;
- sugestões de datasets públicos;
- melhoria dos notebooks;
- criação de exercícios e desafios.

## Fluxo recomendado

1. Abra uma issue descrevendo a proposta.
2. Faça um fork do repositório.
3. Crie uma branch.
4. Envie um pull request.

## Critérios

Contribuições devem preservar:

- clareza didática;
- rigor técnico;
- reprodutibilidade;
- respeito a licenças de terceiros;
- consistência com o escopo panorâmico-aplicado do minicurso.
